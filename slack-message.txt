Just shipped a side project I've been working on — a CLI toolkit called **Omni Skills** that unifies all your AI agent skills across different tools (Claude, Codex, Kimi, Cursor, etc.) into one place.

If you use multiple AI coding tools and your skills/instructions are scattered everywhere, this might save you some headache. It auto-detects your tools, syncs skills, and runs a weekly health check so nothing drifts.

**Try it:** `npm install -g ai-omni-skills`

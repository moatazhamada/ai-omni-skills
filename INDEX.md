# Skills Index

This file is auto-generated by `skills index`.

In the toolkit repo, this index is intentionally minimal. Once you set up your private skills store and run `skills index`, this file will populate with your actual skills ordered by recent usage.

## Getting Started

```bash
skills setup    # Configure your private skills directory
skills index    # Generate your skills index
```

Your private skills index will be created at the root of your configured skills directory.

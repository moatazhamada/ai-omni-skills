# Release Preparation Workflow

A standard pre-release checklist using skills.

## Trigger

When you are ready to release and want to verify quality before committing.

## Steps

1. **lint-and-validate** — Check code quality, formatting, and types
2. **testing-patterns** — Verify all tests pass and coverage is adequate
3. **commit-suggest** — Generate a clean, conventional commit message

## Usage

Ask your AI assistant: "Run the release-prep workflow"

# Feature Development Workflow

A complete workflow for building a new feature using the Spec Kit skill chain.

## Trigger

When you have a new feature idea and want to go from raw requirements to committed code.

## Steps

1. **refine-requests** — Clarify and structure the feature request
2. **speckit-specify** — Write a formal specification
3. **speckit-plan** — Generate an implementation plan
4. **speckit-implement** — Execute the plan and write the code
5. **speckit-git-commit** — Commit all changes with a good message

## Usage

Ask your AI assistant: "Run the feature-dev workflow for adding dark mode"
